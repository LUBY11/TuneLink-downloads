# TuneLink Downloads

This repository hosts release packages for the TuneLink Chrome extension.

## What it does
- Create or join rooms
- Sync YouTube Music playback (host controls)
- Live participants list
- In-room chat

## Usage
1) Download the latest `TuneLink-vX.Y.Z.zip`.
2) Unzip it.
3) Chrome/Edge → `chrome://extensions` → Developer mode → Load unpacked.
4) Select the unzipped folder.

## Notes
- Server URL can be configured in the extension panel.
